# 额外 JVM 参数

ArkPets 的额外 JVM 参数，这些参数一般用于内部使用或调试用途。

目前仅支持在 `.jar` 版本或开发中使用。

### 用法
```
java -D<选项>=<值> -jar ArkPets.jar
```
| 选项                    | 描述                          |
|:----------------------|:----------------------------|
| `arkpets.mactitlebar` | 强制启用 macOS 风格的标题栏。(v4.0.0+) |
